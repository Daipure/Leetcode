當花的總數為奇數時 先拿的會贏
(n+1)/2 可計算奇數的數量
n/2 為偶數的數量

```cpp
class Solution {
public:
    long long flowerGame(int n, int m) {
        long long odd_n = (n+1)/2;
        long long even_n = n/2;
        long long odd_m = (m+1)/2;
        long long even_m = m/2;   

        return odd_n*even_m + even_n*odd_m;     
    }
};
```
