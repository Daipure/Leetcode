一開始奇數個alice win
偶數個 alice拿一個變成奇數 bob 只能拿偶數個 下一回合alice win

```cpp

class Solution {
public:
    bool doesAliceWin(string s) {
        int vow_count = 0;
        unordered_set<char> vowels = {'a', 'e', 'i', 'o', 'u'};

        for (int i = 0; i < s.size(); i++) {
            if (vowels.count(s[i])) {
                vow_count++;
            }
        }

        // 如果一開始母音數就是奇數，Alice 直接清場獲勝。

        // 如果一開始母音數是偶數，Alice 就只刪 1 個母音，把一個「總母音數為奇數」的局面丟給 Bob。

        // Bob 拿到奇數局面，他絕對無法清場，只能繼續掙扎，最後留給 Alice 的還是一個母音數為奇數的局面。
        return vow_count > 0;



    }
};
```
