維護一個不重複的子字串 並去紀錄它的長度  如果要新增的字元會使原本的字串重複 left往右移使其不重複


```cpp

class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        // 維護一個不重複的子字串 並去紀錄它的長度
        // 如果要新增的字元使原本的字串重複 left往左移使其不重複

        unordered_map<int,int>counter;

        int left =  0;
        int res = 0;

        for(int right=0;right<s.size();right++){
            counter[s[right]]++;
            while(counter[s[right]]>1){
                counter[s[left]]--;
                left++;
            }
            res = max(res,right-left+1);
        }
        return res;

    }
};
```
